## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. No build process or dependencies required!

## Customization

### Change Colors
Edit the color variables in `styles.css`:
- Primary gold: `#d4af37`
- Primary dark: `#2d2d2d`

### Update Content
Edit text directly in `index.html`:
- Company name, descriptions
- Service details
- Pricing information
- Contact information

### Add Images
Replace gradient backgrounds with actual images:
```css
background: url('image.jpg') cover center;
```

### Modify Portfolio
Update portfolio items in the HTML section with your actual invitation designs.
